Need a YouTube API Key?
  1. Go to Google Cloud Console
  2. Create a new project (or use existing)
  3. Enable YouTube Data API v3 in the APIs library
  4. Go to Credentials → Create API Key
  5. Copy the key and paste it above
  6. Free tier: 10,000 requests/day (plenty for students)

💡 Don't have a key? Just use our pre-curated videos below—they work great!
